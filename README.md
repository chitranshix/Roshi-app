Most vocabulary tools are flashcard drills — passive, solitary, and easy to abandon. Real retention happens when you're on the spot: explaining a word to someone, getting called out when you're wrong, competing with friends. Roshi is built around that. You dare friends with words you know, they dare you back, and a grumpy turtle judges your definitions.

A social vocabulary game where you dare friends with words you know. They pick the correct sentence, then define the word — Claude grades the definition. Daily word challenges, a leaderboard, traps you can set on levels, and a grumpy turtle mascot who judges you.

**Stack:** Next.js 16 · TypeScript · CSS Modules · Supabase (Postgres + Auth) · Anthropic API (Haiku for grading, Sonnet for word commentary) · Web Push · Vercel
